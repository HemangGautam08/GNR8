# Local Setup
- clone the repo

## env setup
- none for now

## frontend
- cd into FE
  ```cd FE```
- install dependencies
  ```npm i```
- run in dev mode
  ```npm run dev```

## backend
- **Run The Model Prior to starting the BE service**
- cd into BE
  ```cd BE```
- install dependencies
  ```npm i```
- run backend
  ```node index.js```

## Model
- cd into Model
 ```cd Model```
- run Model 
 ```uvicorn main:app --reload```
